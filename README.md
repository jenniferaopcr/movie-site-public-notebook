# movie-site-public-notebook
本仓库用于以笔记形式整理互联网上已公开存在的影视类网站基础信息，重点围绕电影与电视剧相关站点进行系统化记录。通过统一整理方式，对站点名称、访问地址、站点定位说明及基础分类信息进行归纳汇总，方便长期保存与集中查阅。本项目不涉及任何影视资源的存储、播放、下载或解析行为，仅针对公开网络环境中可获取的信息进行整理与记录。所有内容均来源于公开渠道，整理过程保持客观、中立，仅作为资料整理与信息参考用途存在，不构成任何形式的推荐或引导。
